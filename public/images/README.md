# Landing page images

Drop your real photos here using the exact filenames below. The landing page
references these paths directly — no code changes are needed once the files
exist. Until then, a styled placeholder ("نمونه عکس") is shown automatically.

| File                 | Used for                              | Suggested size      |
| -------------------- | ------------------------------------- | ------------------- |
| `menu-coffee.jpg`    | Hot coffee item (phone menu)          | ~400×400 (square)   |
| `menu-cake.jpg`      | Dessert item (phone menu)             | ~400×400 (square)   |
| `menu-iced.jpg`      | Iced drink item (phone menu)          | ~400×400 (square)   |
| `cafe-ambiance.jpg`  | Cafe header banner (phone mockup)     | ~800×320 (landscape)|

Tip: keep files optimized (JPG/WebP, < 200 KB each) for fast loading.
